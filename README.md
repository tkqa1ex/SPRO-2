# SPRO-2: Motor Control & Generator Project

## 🟦 Project Overview
This repository contains the hardware designs and firmware for our university semester project (SPRO-2). The goal of the project is to design and implement a motor control system featuring a custom PCB stack for power management, sensing, and signal generation.

---

## 📂 Repository Structure

### 🔌 [Hardware](./motor-PCB)
The hardware is designed in **KiCad** and is split into three main modules:
* **Control Board:** The "brains" of the operation, handling logic and signal processing.
* **Generator Board:** Responsible for power generation and distribution.
* **Current Sensor Test Board:** A dedicated board for validating the current sensing circuitry before full integration.

### 💻 [Firmware](./Firmware)
*(Coming Soon)*
The firmware will handle the motor control algorithms, sensor data acquisition, and communication logic.
